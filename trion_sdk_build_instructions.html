TRION SDK Build Instructions¶

Please update the DEWETRON TRION API on your system to the latest relase.

The latest release is TRION 5.3.0 from September 2020.

Download it from githubs release section or from ccc.dewetron.com.

Requirements¶

Windows:

  • DEWETRON TRION 5.3.0 or later (https://ccc.dewetron.com/pl/dewe2-series)

  • Microsoft Visual Studio 2017 (or better) with C++ compiler and toolchain (https://visualstudio.microsoft.com/)

  • CMake 3.1 or later (https://cmake.org/download/)

Make sure Visual Studio and CMake are installed and the cmake command can be run from the command prompt.

Linux:

  • Please contact sdk_support@dewetron.com

Quick Summary for Experienced C/C++ Developers¶

  • Start CMakeLists.txt are located in following directories:

    • trion/C

    • trionet/C

    • trionet/CXX

    • trionet/CS

These are used to build the examples.

Step by Step Instructions (C language)¶

The following description assumes that the TRION-SDK repository is cloned to c:\TRION-SDK.

  • Open a command line prompt and change to the directory c:\TRION-SDK\trion\C

  • Create a new directory ‘build’ and change to c:\TRION-SDK\trion\C\build

    mkdir build

    cd build

  • Generate a Visual Studio solution by using the following command:

    cmake ..

    If CMake encounters an error you need to fix the issue and then invoke

    del CMakeCache.txt

    before retrying to ensure a clean run.

  • Open the generated Solution in Visual Studio

    start TRION_SDK_C.sln
