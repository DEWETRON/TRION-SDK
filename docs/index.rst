======================
DEWETRON TRION-SDK 6
======================

**TRION Software Development Kit**

TRION-SDK is a set of libraries and tools that allow to access TRION and TRION3 based measurement
devices:


.. list-table:: 

   * - .. figure:: _img/TRION-series-modules-in-a-row-768x314.jpg
           :alt: TRION devices
           :width: 5in

           TRION and TRION3 modules

     - .. figure:: _img/TRIONet-smallest-power-analyzer.jpg
           :alt: TRIONet and TRIONet3 frontends

           TRIONet and TRIONet3 frontends

     - .. figure:: _img/NEXDAQ-in-a-hand-for-windows-and-linux.jpg
           :alt: NEX[DAQ] devices

           NEX[DAQ] devices




.. note:: TRION-SDK is not a complete measurement application and does not provide one.
