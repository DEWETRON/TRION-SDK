TRION and TRIONet System Overview¶

https://www.dewetron.com/products/data-acquisition-hardware-chassis

A DEWETRON measurement system typically consists of a number of TRION modules and a DEWE2 or DEWE3 enclosure.

The TRIONet frontend is similar to a DEWE2 or DEWE3 enclosure but offers Gigabit LAN and USB3 connectivity.

TRION System¶

TRION measurement system¶

TRION Driver (Kernel Mode Driver)¶

The kernel driver is providing the actual hardware access and the DMA mechanics for data acquisition.

TRION API¶

TRION API is providing a access and progamming interface to an application.

TRION XML¶

TRION XML is a TRION board and system database to be used by the API.

DMA Memory¶

The api automatically allocates the DMA memory necessary for synchronous and asynchonous measurement data. Measurerment data is not copied by the api. The api allows direct access to the buffer by external applications.

Application¶

This is the actual application, using the measurement hardware, by interfacing to it via the API.

Example: DEWETRON Oxygen

TRIONet System¶

TRIONet measurement system¶

TRION Driver (Kernel Mode Driver)¶

The kernel driver is providing the actual hardware access and the DMA mechanics for data acquisition.

TRION API¶

TRION API is providing a access and progamming interface to an application.

TRION XML¶

TRION XML is a TRION board and system database to be used by the API.

DMA Memory¶

The api automatically allocates the DMA memory necessary for synchronous and asynchonous measurement data. Measurerment data is not copied by the api. The api allows direct access to the buffer by external applications.

TRIONet Control Server¶

TRIONet Control Server is running on the TRIONet device and provides services for accessing the TRION API over Ethernet.

TRIONet System Server¶

TRIONet Control Server is running on the TRIONet device and provides servics for device maintainance.

TRIONet API¶

The TRIONet API is an interface compatible API that allows applications to use TRIONet devices similar to TRION boards of a local measurement system.

DMA Memory (local copy)¶

A copy of the TRIONet devices ringbuffer on the host system.
